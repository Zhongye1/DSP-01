# Python PDF题库处理工程指南

## 项目概述

这是一个专门用于处理408计算机统考真题PDF文档的Python工程，能够提取题目、表格和图片，并按照教材章节进行分类整理，最终生成符合题库模板的规范文本格式。

## 技术栈

- **PyMuPDF (fitz)**: 高效处理PDF文档，提取文本、表格和图片
- **pandas**: 处理提取的表格数据
- **Python 3.x**: 主要编程语言

## 安装依赖

```bash
pip install -r requirements.txt
```

主要依赖包：
- `PyMuPDF>=1.24.0`
- `pandas>=2.0.0`
- `requests>=2.25.1`

## 工程结构

```
├── pdf_processor.py              # 通用PDF题库处理器
├── ds_question_processor.py      # 数据结构题库处理器（专门处理408真题中的数据结构部分）
├── requirements.txt             # 项目依赖
├── PYTHON_PDF_PROCESSING_GUIDE.md # 本使用指南
└── 2009年计算机408统考真题.pdf     # 示例PDF文件
```

## 核心功能

### 1. PDFQuestionProcessor (通用处理器)

- 提取文本块、表格和图片
- 识别题目类型（单选题/问答题）
- 按章节对题目进行分类
- 保存为JSON、文本或Excel格式

### 2. DSQuestionProcessor (数据结构处理器)

- 专门处理408真题中的数据结构部分
- 使用数据结构相关关键词进行更精确的识别
- 按照《数据结构（C语言版）第3版》章节进行分类

## 使用方法

### 运行通用PDF处理器

```bash
python pdf_processor.py
```

### 运行数据结构专用处理器

```bash
python ds_question_processor.py
```

## 功能特性

### 文本提取
- 支持按块、行、字提取文本
- 保留段落、题目编号与选项的相对位置
- 高精度提取，特别适合考试类文档

### 表格处理
- 自动识别PDF中的表格
- 转换为pandas DataFrame格式
- 保存为Excel文件

### 图片提取
- 自动提取嵌入的图片
- 保持原始质量
- 按页面分类保存

### 题目识别
- 智能识别单选题和问答题
- 提取选项、参考答案和答案解析
- 按数据结构章节分类

## 输出格式

### JSON格式
- 保存完整的处理结果，便于程序处理

### 文本格式
- 符合题库模板要求的规范文本
- 按章节组织，易于人工审核

### Excel格式
- 提取的表格数据
- 便于数据分析和处理

## 章节分类

按照《数据结构（C语言版）第3版》（严蔚敏、李冬梅、吴伟民编著）的目录进行分类：

1. 第1章 绪论
2. 第2章 线性表
3. 第3章 栈和队列
4. 第4章 树与二叉树
5. 第5章 图
6. 第6章 查找
7. 第7章 排序

## 使用场景

1. **题库建设**: 将历年408真题转换为标准格式
2. **教学辅助**: 为教师提供分类整理的题目资源
3. **学习工具**: 为学生提供按章节分类的练习题
4. **研究分析**: 对题目进行统计分析

## 注意事项

1. 确保PDF文件路径正确
2. 对于扫描版PDF，建议先进行OCR处理
3. 根据实际需求调整正则表达式模式
4. 大文件处理时注意内存使用情况

## 性能优化

- PyMuPDF基于C实现，处理效率高
- 采用块级处理，减少内存占用
- 支持批量处理多页PDF文档
