# 408考研真题处理器

这个工具可以帮助你处理408考研真题PDF文件，并按年份将它们分类到不同的文件夹中。

## 功能特性

- 自动识别PDF文件名中的年份
- 使用MinerU API解析PDF内容
- 将解析结果按年份分类存储
- 分别存储到试题目录(`408qes`)和答案目录(`408ans`)

## 准备工作

1. 确保你的`config.json`文件中有有效的MinerU API令牌
2. 将需要处理的PDF文件放在项目根目录下，文件名中应包含年份（如：`2009年计算机408统考真题.pdf`）

## 使用方法

### 方法一：使用process_exams_by_year.py（推荐）

```bash
python process_exams_by_year.py
```

这个脚本会自动查找当前目录下所有包含年份的PDF文件，使用MinerU的批量上传功能处理它们，并按年份创建子目录进行分类。

### 方法二：使用408ocr.py

```bash
python 408ocr.py
```

这个脚本提供了基本的PDF处理功能。

## 注意事项

- 单个文件大小不能超过200MB，文件页数不超过600页
- 每个账号每天享有2000页最高优先级解析额度
- 因网络限制，确保API调用不受地域限制
- 处理大文件可能需要较长时间，请耐心等待

## API配置

在`config.json`中设置你的API令牌：

```json
{
  "api_base_url": "https://mineru.net/api/v4",
  "api_token": "your_mineru_api_token_here",
  "default_model": "vlm",
  "default_image_model": "wanxiang"
}
```

## 常见问题

1. **无法从文件名提取年份**：请确保PDF文件名中包含年份，格式如"2009年..."或"...2009..."

2. **API调用失败**：检查API令牌是否正确且未过期

3. **处理时间过长**：大文件或高负载时可能需要较长时间，请耐心等待
